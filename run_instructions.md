## 🛠️ Setup Instructions

### Method 1: XAMPP (Recommended)
1. Download fresh XAMPP from: https://www.apachefriends.org/
2. Install and start Apache + MySQL
3. Copy project to: `C:\xampp\htdocs\studentfeedback\`
4. Visit: `http://localhost/studentfeedback/`

### Method 2: Alternative Local Server
1. Install PHP from: https://windows.php.net/download/
2. Add PHP to system PATH
3. Run: `php -S localhost:8000`
4. Visit: `http://localhost:8000`

### Method 3: Online Hosting
1. Upload files to any web hosting service
2. Import database using phpMyAdmin
3. Update `db.php` with hosting credentials
